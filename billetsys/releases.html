billetsys 0.2.0 May 8, 2026
  • New UI frameworks — Frontend built on React + TypeScript with shadcn/ui and Tailwind CSS.
  • Public & private messages — Role-scoped private notes alongside public replies.
  • References & mentions — Cross-reference and mention tickets and articles.
  • CSV import — Bulk-onboard tickets from external sources.
  • Search — Search across ticket numbers and message content.
  • Light/dark mode & captcha — Theme switching and captcha support, including forgot-password.
  • Per-user email format — Each user chooses how notifications are formatted.
