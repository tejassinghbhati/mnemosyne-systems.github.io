orangu
AI coding that runs on your machine.

A terminal-based, locally-hosted AI coding assistant for OpenAI-compatible servers — especially llama.cpp. A focused, offline-first, Git-centric experience with a built-in code reviewer, and no data ever leaving your machine.

$ curl -fsSL https://mnemosyne-systems.github.io/orangu/install.sh | sh
Get started Features GitHub

  • Complete privacy

    Zero telemetry, no cloud uploads. After the model is downloaded, no internet is required.

  • Built-in code review

    Interactive /review and automated /auto_review with per-line, categorised findings.

  • Git-centric workflow

    The full Git loop plus GitHub and GitLab pull requests, comments, and issues — from the prompt.

  • Single native binary

    Written in Rust for fast startup. Workspace-aware tools, agent skills, and multi-project tabs.

billetsys
Support that every role can move faster on.

A modern, open source support ticket system that keeps the whole case — status, ownership, conversation, attachments, and product context — on one record. Five roles share one model, each with a focused screen, built on Quarkus, PostgreSQL, and React.

$ git clone https://github.com/mnemosyne-systems/billetsys.git && cd billetsys && make
Get started Features GitHub

  • Built around roles

    User, Superuser, TAM, Support, and Admin each get a focused view of the same shared ticket model.

  • Conversation in context

    Public and private message threads live on the ticket, with role-scoped visibility and a Markdown editor.

  • Two-way email

    Outgoing notifications plus incoming mail and IMAP/POP3 polling, all feeding the same ticket history.

  • Knowledge & reporting

    A searchable article base, reports on workload and response times, CSV import, and PDF export.
